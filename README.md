This is a Tic-Tac-Toe game project fully functional on webpages i.e. browsers.
It is made fully out of HTML , CSS & DOM.
### Features
- Two-player Tic-Tac-Toe
- Input validation with custom popup
- Win & draw detection
- Replay & restart functionality
- Visual highlight for winning cells

### Future Improvements
- Single-player mode with AI
- Difficulty levels
- Scoreboard
